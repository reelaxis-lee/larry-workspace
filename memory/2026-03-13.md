## Bugs Found + Fixed
1. **Duplicate comments** — same post matched twice due to CSS union selector `[data-id...], .feed-shared-update-v2`. Fixed: single selector + dedup Set by data-id + scope editor/submit to post element + DOM check for existing comments.
2. **Comments not submitting** — wrong submit button selector. Fixed: `button.comments-comment-box__submit-button--cr` (not `button[type="submit"]:has-text("Post")`).
3. **Connections not sending** — modal iframe was blocking clicks on panel approach. Fixed: switched to profile page navigation + `button[aria-label="Open actions overflow menu"]` + `li:has-text("Connect")` dropdown item.
4. **Comment editor bleeding into previous post** — `page.locator('.ql-editor')` was finding still-open editor from previous iteration. Fixed: scoped to `post.locator(...)` + Escape before/after each comment.
