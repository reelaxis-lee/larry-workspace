# Memory — 2026-03-25

## Profile Onboarding System — Built & Live

### What it is
A Claude-powered chat intake app that interviews new account owners to collect everything Larry needs to set up their LinkedIn automation profile. Replaces Darren acting as middleman.

### Live URL
**https://onboarding-nu-lovat.vercel.app**

### How it works
1. Darren sends the URL to a new client
2. Client chats with Claude — interviewed conversationally, one or two questions at a time
3. Claude collects: full name, LinkedIn URL, company, email, timezone/city, offer, ICP, CTA, tone, Sales Nav URL, auto-signature
4. On completion, Claude confirms the summary with the client, then emails the intake JSON to darren@reelaxis.com via Postmark
5. Larry receives the email, creates `profiles/<nickname>/ACCOUNT.md` + `HISTORY.md`, and notifies Darren in #linkedin-updates Slack

### What Darren still does after intake
1. Set Chrome profile path in the new ACCOUNT.md
2. Run `node scripts/setup-profile.js <nickname>` to do the one-time LinkedIn login
3. Confirm profile is added to scheduler
