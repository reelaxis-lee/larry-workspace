## Sales Nav URL Saving — New Standard Process (1:14 PM PT)

When a Sales Nav URL is provided that contains `query=` or `recentSearchParam` (raw/unsaved):
1. Run `scripts/save-search.js <nickname> "<url>" "<Search Name>"`
2. Script opens Playwright with the profile's browser context
3. Navigates to the URL, clicks the "Save search" toggle, fills in a descriptive name
4. Captures `savedSearchId` from redirect URL
5. Prints clean `savedSearchId=XXXXXXXXXX` URL + exact ACCOUNT.md line to paste
6. Update ACCOUNT.md with the clean URL

If URL already has `savedSearchId=` — use directly, no script needed.

**Why:** Raw query URLs include a `sessionId` parameter that expires. Saved searches are permanent.

Script location: `scripts/save-search.js` — reusable for any profile/nickname.
